The ACFT Calculator is a project designed for individual soldiers and NCOs to allow easy calculation and tracking of ACFT scores. There is a full release available for download and use in the repository.


C Client/Server

Repository

My first project and experience writting in C, this simple client/server application allows a user to send files and messages to and from a server on their private network. This was created primarily to be ported and expanded to introduce client/server architecture to the ACFT Calculator.


GoSpicyRamen

Repository

GoSpicyRamen is the continuation of one of my first projects - SpicyRamen, a Discord bot written in Java to begin learning web scraping, data collection, and data management. GoSpicyRamen is the same idea, but written in Go and features the ability to send queries to YouTube, Reddit, and UrbanDictionary, collect the results, and display the information in an easy to use paginator.


Automated AD Audit

Repository

One of my first projects in Go and one of my first projects I used professionally, this simple program allows one to easily audit changes between two documents. The complete version is unavailable to the public, but it allowed me to easily pull new records from Active Directory and compare them with records saved from the previous audit. This allowed me to ensure that all onboarded and offboarded employees were properly tracked and their equipment properly handled.
